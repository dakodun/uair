RenderBatchData
Overview
A RenderBatchData object holds converted render data from a Renderable that has been added to a Render Batch, for use in a VBO.
Data Members
Public:
std::vector<VBOVertex> mVertData
The vertex data that forms the Renderable that this data was converted from.
std::vector<VBOIndex> mIndData
A std::vector containing the indices used to render the Renderable that this data was converted from.
GLuint mTexID
The id of the texture used by the Renderable that this data was converted from.
std::string mTag
The tag (type) of the Renderable that this data was converted from.
